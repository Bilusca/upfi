
# UPFI

Project created with Next and FaunaDB to upload images.

## Demonstration

![Project running](/project-images/upfi.gif)


## Running local

Clone project

```bash
  git clone https://github.com/Bilusca/upfi.git
```

Install dependecies

```bash
  yarn
```

Running app

```bash
  yarn dev
```

## Stack

**Front-end:** NextJS, FaunaDB, ChakraUI, React Query
